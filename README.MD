# 🚀 Technologies

This project was built using the following technologies:

## 📦 Dependencies

- [TypeScript][typescript-npm] — JavaScript superset for scalable applications
- [TSX][tsx-npm] — Run TypeScript files directly in Node.js
- [TSUP][tsup-npm] — Fast TypeScript bundler powered by esbuild
- [@types/node][@types-node-npm] — Type definitions for Node.js

# 📁 Project Files

- `.gitignore` — Ignores unnecessary files and folders
- `.env` — Environment variables configuration
- `tsconfig.json` — TypeScript compiler settings

---

# ⚡ Available Scripts

| Script | Description |
|--------|-------------|
| `npm run dist` | Compiles TypeScript files into the `dist` folder |
| `npm run start:dev` | Starts the project in development mode |
| `npm run start:watch` | Starts development mode with auto-reload |
| `npm run start:dist` | Builds and runs the production version |
